# My Git

## Getting started

Perhaps you have to do this first:

```
sudo apt install git-all
```

1.

```
sh ./install.sh
```

2. On Linux use:

```
sh ./install_linux.sh
```

2. On MacOS use:


```
sh ./install_macos.sh
```

Run gitinit.sh in that directory you want to start using git.

There are new commands available like:

```
git l
git days
git dayscount
git stat
git authors
git create-file
git simple-commit
```
