# My Git

## Getting started

On Linux use:
```
sh ./install_linux.sh
```

On MacOS use:
```
sh ./install_macos.sh
```

Run gitinit.sh in that directory you want to start using git.

When you have added gitalias to your global git config file, there are new
commands available like:
git l
git days
git dayscount
git stat
git authors
git create-file
git simple-commit
